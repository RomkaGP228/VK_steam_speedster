# steam_speedster
Скрипт для анализа скорости загрузки приложений steam с помощью его всторенных логов. Разработано под Windows.

```shell
git clone https://github.com/RomkaGP228/steam_speedster.git
cd un_grande_compo

# Linux or Mac
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 main.py

# Windows
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
